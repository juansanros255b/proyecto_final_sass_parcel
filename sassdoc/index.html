    px-to-rem

    @function px-to-rem($px, $base: 16) { ... }

    Description

    Convierte un valor en píxeles a su equivalente en rem. El cálculo asume una base de 16px por defecto (estándar de los navegadores). Cambia $base si el font-size del <html> es diferente.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $px

    Valor en píxeles a convertir (sin unidad o con px)

    		Number— none
    $base

    Tamaño base en píxeles del elemento raíz

    		Number16

    Returns

    Length —

    El valor equivalente en rem

    Example

    Convertir 24px a rem con base estándar

    .heading {
  font-size: px-to-rem(24px); // → 1.5rem
}

    Convertir con base personalizada de 10px

    .label {
  font-size: px-to-rem(14px, 10); // → 1.4rem
}

    respond-to

    @mixin respond-to($breakpoint) { ... }

    Description

    Genera una media query min-width a partir de un nombre de breakpoint predefinido. Los valores exactos están definidos en _variables.scss como $breakpoints. Si la clave no existe emite un @warn y no genera ninguna regla.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    Nombre del breakpoint: sm (576px), md (768px), lg (992px) o xl (1200px)

    		String— none

    Example

    Cambiar layout en tablet

    .card {
  width: 100%;
  @include respond-to("md") { width: 50%; }
  @include respond-to("lg") { width: 33%; }
}

    Requires

    • [variable] breakpoints
    • [variable] breakpoints

    text-truncate

    @mixin text-truncate($lines: 1) { ... }

    Description

    Trunca el texto de un elemento con puntos suspensivos (…). Con $lines: 1 usa la técnica white-space: nowrap (compatible con todos los navegadores). Con $lines > 1 usa -webkit-line-clamp (compatible con navegadores modernos).

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $lines

    Número máximo de líneas visibles antes del truncado

    		Number1

    Example

    Truncar a una sola línea

    .card__title {
  @include text-truncate();
}

    Truncar a tres líneas

    .card__excerpt {
  @include text-truncate(3);
}

    transition

    @mixin transition($props: all, $duration: 0.3s, $easing: ease) { ... }

    Description

    Aplica una transición CSS configurable en propiedades, duración y curva de animación. Úsalo en cualquier elemento que necesite animación suave en hover, focus o estado activo.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $props

    Propiedad o lista de propiedades a animar

    		Stringall
    $duration

    Duración de la transición

    		Time0.3s
    $easing

    Función de temporización (ease, ease-in, ease-out, linear…)

    		Stringease

    Example

    Transición solo en el color de fondo

    .btn {
  @include transition(background-color, 0.2s, ease-in);
}

    Transición en varias propiedades

    .card {
  @include transition(transform box-shadow, 0.25s);
}

    auto-grid

    @mixin auto-grid($min-col: 280px, $gap: 1.5rem) { ... }

    Description

    Crea un grid CSS responsivo con columnas de ancho mínimo configurable que se ajustan automáticamente al espacio disponible (auto-fit).

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $min-col

    Ancho mínimo de cada columna antes de reflow

    		Length280px
    $gap

    Espacio entre celdas (filas y columnas)

    		Length1.5rem

    Example

    Grid de features con columnas de mínimo 250px

    .features__grid {
  @include auto-grid(250px, 1rem);
}

    Grid de testimonios con columnas más anchas

    .testimonials__grid {
  @include auto-grid(300px, map-get($spacing, "lg"));
}
